# lot_drawing-system-zh_TW
用來抽籤的小程式

# 編譯
1. 確認make和編譯器已安裝在你的機器上
2. 在終端執行  
   `make CXX="你的C++編譯器"`  
     
   輸出的可執行檔名稱預設為lotdraw（或lotdraw.exe，若你使用的是Windows）  
   你可以在執行make時覆寫EXEC變數來自定義可執行檔的名稱

# 執行
在終端輸入可執行檔的名稱就能啟動程式。  
詳細的使用說明請參照[Wiki](https://github.com/jeang-bo-yuan/lot_drawing-system-zh_TW/wiki)
